# Plan: Exploratory Analysis of morcom26_cux2mice P26_ENs.h5ad

## Context
New dataset: excitatory neurons (ENs) from CUX2 mouse experiment (morcom26). The user wants an exploratory script covering cell metadata (cell type abundance, sample/condition breakdown) and basic QC metrics. This follows the same workflow as existing scripts in `scripts_data_wrangling/`.

**Dataset summary:**
- 27,970 cells × 29,178 genes
- 6 samples: cx-WT-1/2/3, cx-Null-1/2/3
- 2 conditions: WT vs Null (CUX2 knockout)
- 21 cell types (EN subtypes + "Other-mix" categories)
- QC columns: `total_counts`, `n_genes_by_counts`, `pct_counts_mt`, `pct_counts_ribo`, `doublet_scores`, `predicted_doublets`
- Cell cycle: `S_score`, `G2M_score`

## Output File
**Script:** `scripts_data_wrangling/01.explore_morcom26_cux2mice.py`  
**Figure dir:** `local_data/fig/morcom26_cux2mice/`

## Figures (4 panels/figures total)

### Fig 1: Cell type abundance (1 panel)
- Horizontal barplot of cell counts per `celltype`, sorted descending
- Save: `01.celltype_abundance.png`

### Fig 2: Cell type composition by sample and condition (2 subplots)
- Left: stacked bar per sample (x=sample, colors=celltype)
- Right: stacked bar per condition (x=Condition, colors=celltype)
- Normalize to fraction (0–1) for comparability
- Save: `01.celltype_composition.png`

### Fig 3: QC metrics distributions per sample (4-panel violin)
- `total_counts`, `n_genes_by_counts`, `pct_counts_mt`, `doublet_scores`
- x=samples, colored by Condition (WT vs Null)
- Save: `01.qc_metrics.png`

### Fig 4: QC scatter — total_counts vs n_genes (1 panel)
- Scatter colored by `Condition`; doublets marked with distinct alpha
- Helps identify outlier cells and doublet distribution
- Save: `01.qc_scatter.png`

## Key Patterns to Follow
