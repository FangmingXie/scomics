# Plan: scripts/astro/04.num_archetype_p56.py

## Context

Create an archetype number selection script for P56 astrocyte cells, following the same pattern as `scripts/cheng22/04.num_archetype_cheng22.py`. The astrocyte dataset spans multiple developmental ages; this script focuses on the P56 timepoint. The output is a metrics PNG and an interactive per-region archetype overlay HTML.

---

## Critical Files

| File | Role |
|------|------|
| `scripts/astro/04.num_archetype_p56.py` | **New file to create** |
| `scripts/cheng22/04.num_archetype_cheng22.py` | Template to follow |
| `scripts/common.py` | Provides `select_hvg()`, `run_noc_sweep()` |
| `scripts/viz.py` | Provides `save_metrics_plot()`, `scatter_per_group_html()` |
| `local_data/source/gao25/DevVIS_scRNA_AstroTE.h5ad` | Input data |

---

## Key Differences from cheng22/04

| Item | cheng22/04 | astro/04 |
|------|-----------|----------|
| Input file | `local_data/processed/cheng22/cheng22.h5ad` | `local_data/source/gao25/DevVIS_scRNA_AstroTE.h5ad` |
| Age filter | none (all cells) | filter to `Age == 'P56'` |
| X format | sparse (`.todense()`) | sparse (`.toarray()`) |
| Depth column | `adata.obs['n_counts']` | compute from `X.sum(axis=1)` (no explicit depth col) |
| Types column | `adata.obs['Type']` | `adata.obs['cluster_label']` |
| Groups/replicates | `adata.obs['sample']` | `adata.obs['donor_name']` (6 donors, 694–1116 cells each) |
| Output prefix | `04.num_archetype_cheng22_*` | `04.num_archetype_p56_*` |

## Implementation

```python
# scripts/astro/04.num_archetype_p56.py
import os, sys
import numpy as np
import matplotlib.pyplot as plt
import anndata as ad
import scipy.sparse as sp

sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
from common import select_hvg, run_noc_sweep
from viz import save_metrics_plot, scatter_per_group_html

from scomics.main import SCA
from scomics.utils import norm

SCRIPTS_DIR       = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
PROJECT_ROOT      = os.path.dirname(SCRIPTS_DIR)
INPUT_FILE        = os.path.join(PROJECT_ROOT, 'local_data', 'source', 'gao25', 'DevVIS_scRNA_AstroTE.h5ad')
FIG_DIR           = os.path.join(PROJECT_ROOT, 'local_data', 'fig', 'astro')
FIG_NOC_METRICS   = os.path.join(FIG_DIR, '04.num_archetype_p56_metrics.png')
FIG_NOC_ARCHETYPES_REP = os.path.join(FIG_DIR, '04.num_archetype_p56_interactive_rep.html')

P56_AGE_VAL = 'P56'
NDIM        = 10
NOC_MIN     = 2
NOC_MAX     = 6
NREPEATS    = 10
N_TOP_GENES = 2000

# load and filter to P56
adata = ad.read_h5ad(INPUT_FILE)
adata = adata[adata.obs['Age'] == P56_AGE_VAL].copy()
print(f'P56 cells: {adata.shape[0]}')

x = adata.X.toarray() if sp.issparse(adata.X) else np.array(adata.X)
depths = x.sum(axis=1)
types  = adata.obs['cluster_label'].values
donors = adata.obs['donor_name'].values

hvg_mask = select_hvg(x, depths, N_TOP_GENES)
xn  = norm(x[:, hvg_mask], depths)
sca = SCA(xn, types)
sca.setup_feature_matrix(method='data')

os.makedirs(FIG_DIR, exist_ok=True)
noc_grid = np.arange(NOC_MIN, NOC_MAX + 1)

ev_grid, av_grid, av_rep_grid, xp_grid, aa_grid, aa_reps_grid = run_noc_sweep(
    sca, noc_grid, NDIM, NREPEATS, donors)

cycle = plt.rcParams['axes.prop_cycle'].by_key()['color']
donor_to_color = {d: cycle[i % len(cycle)] for i, d in enumerate(np.unique(donors))}

save_metrics_plot(noc_grid, ev_grid, av_grid, av_rep_grid,
                  NDIM, f'Archetype selection P56 (NDIM={NDIM})', FIG_NOC_METRICS)

scatter_per_group_html(noc_grid, ev_grid, av_rep_grid, xp_grid, aa_reps_grid,
                       donors, donor_to_color,
                       f'Per-donor archetype overlay P56 (NDIM={NDIM})',
                       FIG_NOC_ARCHETYPES_REP)
```
