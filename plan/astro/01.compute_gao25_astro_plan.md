# Plan: Astrocyte Developmental Analysis Scripts

## Context
Create one compute script in `scripts/astro/` to analyze the gao25 astrocyte development dataset (`DevVIS_scRNA_AstroTE.h5ad`, 32,116 cells × 32,285 genes). It runs both the standard analysis and the P56 projection in a single script, saving both result TSVs.

**Key data facts:**
- `Age`: 31 unique developmental ages, but only 25 have >100 cells (all postnatal: P0–P56)
- The 6 embryonic ages (E13.5–E18.5) are dropped — too few cells (1–93 per age)
- `timepoint`: ['E11-E18', 'P0-P12', 'P13-P28', 'WB'] (broad bins)
- `cluster_label`: 5 clusters, `subcluster_label`: 25 subclusters
- X is sparse CSR
- P56 cells identified via `Age == 'P56'` (5,309 cells — largest group)

---

## Script: `scripts/astro/01.compute.py`

**Purpose:** All computation — standard analysis (PCA + UMAP + KMeans) AND P56-anchored projection — in one script. Shared normalization step to avoid loading the file twice.

### Outputs
- `local_data/res/gao25/01.standard_results.tsv` — UMAP coords, PC coords, cluster labels, metadata
- `local_data/res/gao25/01.p56_projection_results.tsv` — PC coords in P56 PCA space, metadata

### File paths / config (top of script)
```python
PROJECT_ROOT  = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
INPUT_FILE    = os.path.join(PROJECT_ROOT, 'local_data', 'source', 'gao25', 'DevVIS_scRNA_AstroTE.h5ad')
OUT_RES_DIR   = os.path.join(PROJECT_ROOT, 'local_data', 'res', 'gao25')
OUT_STANDARD  = os.path.join(OUT_RES_DIR, '01.standard_results.tsv')
OUT_P56_PROJ  = os.path.join(OUT_RES_DIR, '01.p56_projection_results.tsv')

MIN_CELLS_PER_AGE = 100
N_HVG         = 2000
N_PCS         = 50
N_NEIGHBORS   = 15
N_CLUSTERS    = 10
P56_AGE_VAL   = 'P56'
METADATA_COLS = ['Age', 'timepoint', 'age_bin', 'cluster_label', 'subcluster_label', 'roi', 'sex']
```

### Shared normalization (run once, reused by both analyses)
```python
adata = ad.read_h5ad(INPUT_FILE)

# filter to ages with >100 cells (keeps 25 postnatal ages, P0-P56; drops 6 embryonic ages)
age_counts = adata.obs['Age'].value_counts()
valid_ages = age_counts[age_counts > MIN_CELLS_PER_AGE].index
adata = adata[adata.obs['Age'].isin(valid_ages)].copy()

X = adata.X.toarray().astype(np.float32)
depths = X.sum(axis=1, keepdims=True)
depths = np.where(depths == 0, 1, depths)
X_norm = np.log1p(X / depths * 1e4)        # CP10k + log1p, reused below
```

### Part A — Standard analysis (all filtered cells)
```
HVG by variance on X_norm (top N_HVG) →
StandardScaler.fit_transform →
PCA(N_PCS, random_state=0) →
UMAP(n_neighbors=N_NEIGHBORS, random_state=0) →
KMeans(n_clusters=N_CLUSTERS, random_state=0, n_init='auto') on X_pca →
save 01.standard_results.tsv
```
Columns: `cell_id` (index), `UMAP1`, `UMAP2`, `PC1`–`PC50`, `kmeans_cluster`, + `METADATA_COLS`

### Part B — P56 projection (reuses X_norm)
```
p56_mask = (adata.obs['Age'] == P56_AGE_VAL).values
HVG indices from variance on X_norm[p56_mask] →
StandardScaler.fit on X_norm[p56_mask, hvg_idx] →
PCA(N_PCS).fit on p56 scaled →
scaler.transform + pca.transform on ALL filtered cells →
save 01.p56_projection_results.tsv
```
Columns: `cell_id` (index), `PC1`–`PC50`, + `METADATA_COLS`

### Key notes
- `X_norm` is computed once and shared between Part A and Part B (no duplicate I/O)
- HVG sets for Part A and Part B are computed independently (different variance sources)
- `n_init='auto'` for KMeans; `random_state=0` for PCA, UMAP, KMeans
- `os.makedirs(OUT_RES_DIR, exist_ok=True)` before saving
