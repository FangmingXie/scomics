# Plan: Add `drop_pcs` support and new P56 no-PC2 script

## Context
Archetypal analysis of P56 astrocytes (Gao et al. 2025) uses PCA projection before running PCHA. PC2 sometimes captures a batch/technical signal rather than biology, so analysts want to drop specific PCs (e.g. PC2) before the archetype computation. This is expected to be a recurring need, so it should be a first-class parameter in the core library, not a script-level hack.

---

## Approach: Replace `skip_pc1` with `drop_pcs` throughout the stack

The existing `skip_pc1` boolean is already a special-case of "drop one specific PC". Replace it with a general `drop_pcs` parameter (list of 0-based PC indices, default `None` = drop last PC = current default behavior). This is a clean breaking change; no backward compat needed per CLAUDE.md.

---

## Changes

### 1. `src/scomics/utils.py`

**`proj(x_norm, ndim, method='PCA', drop_pcs=None)`**
- `drop_pcs=None` → fit `ndim+1` components, keep first `ndim` (drop last) — same as before
- `drop_pcs=[1]` (or any list) → fit `ndim + len(drop_pcs)` components, then `np.delete(x_proj, drop_pcs, axis=1)`

```python
def proj(x_norm, ndim, method='PCA', drop_pcs=None):
    n_fit = ndim + (len(drop_pcs) if drop_pcs is not None else 1)
    if method == 'PCA':
        pca_model = PCA(n_components=n_fit)
        x_proj = pca_model.fit_transform(x_norm)
    else:
        raise ValueError('methods other than PCA are not implemented...')
    if drop_pcs is not None:
        x_proj = np.delete(x_proj, drop_pcs, axis=1)
    else:
        x_proj = x_proj[:, :-1]   # drop the extra last component
    return x_proj, pca_model
```

**`proj_transform(x_norm, pca_model, drop_pcs=None)`**  — same logic, no fitting.

---

### 2. `src/scomics/main.py`

Replace `skip_pc1=False` with `drop_pcs=None` in all methods that call `proj` / `proj_transform`:
- `proj_and_pcha(self, ndim, noc, drop_pcs=None, **kwargs)` — store `self.drop_pcs_ = drop_pcs`
- `bootstrap_proj_pcha(self, ndim, noc, ..., drop_pcs=None, ...)` — pass `drop_pcs` to inner `proj()` calls
- `t_ratio_test(self, ndim, noc, nrepeats=10, drop_pcs=None, **kwargs)` — pass through
- `pcha_on_subset(self, mask, noc, drop_pcs=None, **kwargs)` — pass to `proj_transform()`

---

### 3. `scripts/common.py`

`run_noc_sweep(sca, noc_grid, ndim, nrepeats, groups, drop_pcs=None)`
- Pass `drop_pcs` to: `sca.proj_and_pcha(ndim, noc, drop_pcs=drop_pcs)`, `sca.bootstrap_proj_pcha(..., drop_pcs=drop_pcs)`, `sca.pcha_on_subset(gmask, noc, drop_pcs=drop_pcs)`
