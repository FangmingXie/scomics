# Plan: Script 19 — Archetype Scores from P56 Marker Genes

## Context
Scripts 17/18 define 4 archetypes at P56 via PCHA, then relay labels to younger ages via sequential kNN. The goal here is to go beyond discrete labels and compute a continuous **archetype score** for every cell at every postnatal age, grounded in P56 gene expression. Each archetype gets a score = mean z-scored log1p expression over its top 10 marker genes.

## New File
`scripts/astro/19.archetype_scores.py`

### Steps
1. **Load data**: read h5ad, subset to postnatal cells (ages starting with 'P')
2. **HVG selection**: `select_hvg(x[p56_mask], depths[p56_mask], N_TOP_GENES)` — same as script 17
3. **Normalize**: `xn = norm(x[:, hvg_mask], depths)` for all postnatal cells
4. **log1p**: `xl = np.log1p(xn)` — (n_postnatal, n_hvg)
5. **Marker genes at P56**: Load P56 archetype labels from `17.labels_all_ages.parquet` (column `archetype`). For each archetype k:
   - `mean_in  = xl_p56[arch == f'Arch{k+1}'].mean(axis=0)`
   - `mean_out = xl_p56[arch != f'Arch{k+1}'].mean(axis=0)`
   - `top_idx  = argsort(mean_in - mean_out)[::-1][:N_ARCHETYPE_GENES]`
   - Print top gene names
6. **Z-score**: standardize each gene column across all postnatal cells:
   `zl = (xl - xl.mean(axis=0)) / (xl.std(axis=0) + 1e-8)` — (n_postnatal, n_hvg)
7. **Archetype scores**: for each cell, `score_k = zl[:, top_idx_k].mean(axis=1)`
8. **Save parquet**: merge PC coords + donor + age + archetype labels (from script 17 parquet) with score columns `Arch1_score … Arch4_score` → `PARQUET_SCORES`
9. **Visualize**: call `gene_expr_scatter_html` with:
   - `xp = df[pc_cols].values`
   - `gene_vals = {'Arch1_score': scores[:,0], ..., 'Arch4_score': scores[:,3]}`
   - `aa = df_vertices.values.T[:3, :]`  (same as script 18)
   - `panels=PANELS, panel_3d=PANEL_3D`
   - colorbar_title = 'archetype score (z)'
