# Plan: Script 23 — Mean Normalized Gene Expression Boxplot by Sample

## Context

Script 22 (`22.boxplot_Kcnq5_by_sample.py`) plots log2(CP10k+1) boxplots for individual genes per sample with a dropdown. The new script plots a single summary score per cell: the mean across genes of per-gene expression normalized to WT baseline (by subtraction in log space).

## Normalization

```
norm_g(cell) = log2cp10k_g(cell) - mean_{WT cells}(log2cp10k_g)
score(cell)  = mean_g(norm_g(cell))
```

WT cells will center near 0; Null cells will show deviation.

## Implementation Steps

2. Update `OUT_HTML` to `23.boxplot_mean_norm_expr_by_sample.html`.
3. Compute per-gene WT means on the GENES subset:
   ```python
   wt_mask = (adata.obs[CONDITION_COL] == 'WT').values
   gene_idx = [list(adata.var_names).index(g) for g in GENES]
   logcpm2_genes = logcpm2[:, gene_idx]
   wt_means = logcpm2_genes[wt_mask].mean(axis=0)
   ```
4. Subtract WT means and average across genes:
   ```python
   norm_genes = logcpm2_genes - wt_means[np.newaxis, :]

   ```
