## Context
The user wants a new analysis script based on `scripts/03.num_archetype_rep.py` that runs archetype number selection on the Dombrowski 2025 fly dataset, analyzing each (cell type × age group) combination separately.

## Dataset: dombrowski25_fly.h5ad
- Path: `local_data/processed/dombrowski25_fly/dombrowski25_fly.h5ad`
- 6003 cells × 17824 genes
- X: dense numpy.ndarray (no `.todense()` needed)
- Key obs columns:
  - `nCount_RNA` — depth (replaces `n_counts`)
  - `type1` — cell type: LC4, LPLC1, LPLC2, X (skip X)
  - `orig.ident` — age: APF_48h, APF_72h, APF_96h (used as stratification dimension)
  - `genotype` — sample/replicate (replaces `sample` from script 03)
  - `group` — early/late (not used for stratification)

# HVG selection: normalize+log on a copy to get gene mask

del adata_tmp
