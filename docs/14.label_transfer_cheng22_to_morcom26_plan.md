# Plan: Label Transfer from Cheng22 to Morcom26

## Context
Transfer fine-grained L2/3 subtype labels (L2/3_A, L2/3_B, L2/3_C) from the Cheng22 mouse cortex reference dataset to the Morcom26 CUX2 mice EN-L2-3-CTX & EN-L2-mix cells. The morcom26 dataset has broad labels but not L2/3 subtypes; cheng22 has them. This enables studying which L2/3 subtypes map to EN-L2-3-CTX vs EN-L2-mix.

## Data Summary
- **Reference**: `local_data/source/cheng22/cheng22.h5ad`
  - 4044 cells × 286 genes
  - `obs['Type']`: `['L2/3_A', 'L2/3_B', 'L2/3_C']`
  - `obs['n_counts']`: sequencing depth
- **Target**: `local_data/source/morcom26_cux2mice/P26_EN-L2-3-CTX_EN-L2-mix.h5ad`
  - 7556 cells × 29178 genes
  - `obs['celltype']`: `['EN-L2-3-CTX', 'EN-L2-mix']`
  - `obsm['X_umap']`: pre-computed UMAP (reuse for visualization)
  - `obs['total_counts']`: sequencing depth

## Approach: kNN Label Transfer on Shared Genes

1. Find shared genes between cheng22 (286 genes) and morcom26 (29178 genes)
2. Normalize both datasets: CP10k + log1p on shared genes
3. Fit `StandardScaler` on reference, apply same transform to target
4. Fit PCA on reference normalized matrix
5. Transform target cells into the same PCA space
6. Train `KNeighborsClassifier` on reference PCA coords with `Type` labels
7. Predict labels + per-class probabilities for target cells
8. Save TSV results and two visualizations

### Config
```python
REF_CELLTYPE_COL  = 'Type'
REF_DEPTH_COL     = 'n_counts'
TARGET_DEPTH_COL  = 'total_counts'
N_PCS             = 20
N_KNN             = 15
MARKER_SIZE       = 3
MARKER_OPACITY    = 0.6
```

### Algorithm
```python
# Shared genes
shared_genes = ref.var_names.intersection(target.var_names)
if len(shared_genes) < 50:
    raise ValueError(f"Only {len(shared_genes)} shared genes — too few for label transfer")

# Normalize: CP10k + log1p
X_ref    = normalize(ref[:, shared_genes].X,    ref.obs[REF_DEPTH_COL].values)
X_target = normalize(target[:, shared_genes].X, target.obs[TARGET_DEPTH_COL].values)

# Scale: fit on reference, apply to target
scaler = StandardScaler()
X_ref_scaled    = scaler.fit_transform(X_ref)
X_target_scaled = scaler.transform(X_target)

# PCA: fit on reference, transform both
pca = PCA(n_components=N_PCS, random_state=0)
X_ref_pca    = pca.fit_transform(X_ref_scaled)
X_target_pca = pca.transform(X_target_scaled)

# kNN classifier trained on reference
knn = KNeighborsClassifier(n_neighbors=N_KNN, metric='euclidean')
knn.fit(X_ref_pca, ref.obs[REF_CELLTYPE_COL].values)
pred_labels = knn.predict(X_target_pca)
pred_proba  = knn.predict_proba(X_target_pca)
```

### Output TSV columns
