# Plan: DEG Analysis — EN-L2-3-CTX, WT vs Null, LMM with sample random effect

## Context
Differential expression analysis of CUX2 knockout (Null) vs WT for the EN-L2-3-CTX subtype in the morcom26 dataset. Uses a linear mixed model (LMM) via statsmodels to account for sample-level variation as a random effect. Significant genes (|log2FC| > 1, FDR < 0.05) saved as a TSV.

## Output
- **Script:** `scripts_data_wrangling/02.deg_morcom26_cux2mice_ENCTX.py`
- **Output TSV:** `local_data/res/morcom26_cux2mice/EN-L2-3-CTX_WT_vs_Null_sig.tsv`

## Key Variables / Paths
```python
PROJECT_ROOT = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
INPUT_FILE   = os.path.join(PROJECT_ROOT, 'local_data', 'raw', 'morcom26_cux2mice', 'P26_ENs.h5ad')
OUT_DIR      = os.path.join(PROJECT_ROOT, 'local_data', 'res', 'morcom26_cux2mice')
OUT_FILE     = os.path.join(OUT_DIR, 'EN-L2-3-CTX_WT_vs_Null_sig.tsv')

CELLTYPE_COL  = 'celltype'
SAMPLE_COL    = 'samples'
CONDITION_COL = 'Condition'
TARGET_CELL   = 'EN-L2-3-CTX'
REF_CONDITION = 'WT'   # reference level (intercept); Null is the contrast
LOG2FC_THRESH = 1.0
FDR_THRESH    = 0.05
MIN_EXPR_FRAC = 0.1    # pre-filter: keep genes expressed in ≥10% of cells
```

## Implementation Steps

### 1. Load & subset
- `sc.read_h5ad(INPUT_FILE)`
- Filter `adata.obs[CELLTYPE_COL] == TARGET_CELL` → subset AnnData

### 2. Normalize expression
- Use raw counts in `adata.X`; apply CP10k + log1p normalization in-memory (do NOT overwrite adata.X in place)
- Result: dense numpy matrix `logcpm` of shape (n_cells, n_genes)

### 3. Pre-filter genes
- Keep genes expressed (count > 0) in ≥ `MIN_EXPR_FRAC` of cells to reduce from ~29k genes to a manageable set
- Print how many genes pass filter

### 4. Set up condition encoding
- Map `Condition`: `'WT'` → 0, `'Null'` → 1 (so coefficient = Null − WT, i.e., positive = up in Null)
- Build a per-cell metadata DataFrame with columns: `condition_code`, `sample`

### 5. LMM loop over genes (statsmodels MixedLM)
```python
import statsmodels.formula.api as smf
from statsmodels.stats.multitest import multipletests

results = []
for i, gene in enumerate(filtered_genes):
    df = meta_df.copy()
    df['expr'] = logcpm[:, i]
    try:
        model = smf.mixedlm("expr ~ condition_code", data=df, groups=df['sample'])
        fit = model.fit(reml=True, method='lbfgs')
        coef  = fit.params['condition_code']        # log2FC proxy (log space diff)
        pval  = fit.pvalues['condition_code']
    except Exception:
        coef, pval = np.nan, np.nan
    results.append({'gene': gene, 'coef': coef, 'pval': pval})
```
- Print progress every 500 genes

### 6. Compute log2FC and FDR
- `log2FC` = `coef / np.log(2)` (converts natural-log diff to log2)
- FDR via Benjamini-Hochberg: `multipletests(pvals, method='fdr_bh')`
- Assemble results DataFrame with columns: `gene, log2FC, pval, fdr`

### 7. Filter & save
- Keep rows where `fdr < FDR_THRESH` and `abs(log2FC) > LOG2FC_THRESH`
- Sort by `fdr` ascending
- `os.makedirs(OUT_DIR, exist_ok=True)`
- Save: `sig_df.to_csv(OUT_FILE, sep='\t', index=False)`
- Print number of significant genes
